# CRUD Users (Study Project)

This is a small **CRUD project built with Spring Boot**, created purely for learning purposes.

The main goal was to practice:
- Basic REST APIs
- Controllers and service layers
- Spring Data JPA
- Simple database interactions

This project is **not production-ready** and does not aim to follow all best practices.

---

## Tech Stack
- Java
- Spring Boot
- Spring Web
- Spring Data JPA
- Lombok

---

## Features
- Create users
- Find a user by email
- Update a user by ID
- Delete a user by email
